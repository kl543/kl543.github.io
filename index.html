Selected Projects

Feedback Temperature Controller for MBE Power Supply
Python · SCPI · PID/PI tuning · Logging

Implemented computer control of a high-temperature source with adjustable ramp/hold profiles and safety interlocks. Added data logging and simple system ID to tune gains from step responses; reduced manual operation time during growth.
